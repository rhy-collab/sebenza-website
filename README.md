# Sebenza Consulting — website

A static marketing site for Sebenza Consulting, a governance advisory practice.

## Structure

- `index.html` — home
- `approach.html` — services and how engagements run
- `insights.html` — short papers / thought leadership
- `about.html` — about the practice
- `contact.html` — contact details
- `assets/` — stylesheet, logo, favicon

No build step required — this is a plain static site, ready to deploy as-is (e.g. on Vercel with zero configuration).
